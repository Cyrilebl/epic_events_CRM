### Terminal Colors

| Message Type           | Color                                |
| ---------------------- | ------------------------------------ |
| **Prompt text**        | 🟣 Magenta (fg="magenta", bold=True) |
| **User input prompts** | 🔵 Blue (fg="blue", bold=True)       |
| **Success messages**   | ✅ Green (fg="green", bold=True)     |
| **Errors**             | ❌ Red (fg="red", bold=True)         |
| **System status**      | 🟡 Yellow (fg="yellow", bold=True)   |
